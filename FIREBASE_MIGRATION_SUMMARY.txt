│                    🆘 TROUBLESHOOTING                                    │
└──────────────────────────────────────────────────────────────────────────┘

Problem: "Firebase is not defined"
→ Make sure you're accessing via localhost:8000 (not file://)

Problem: "User not found"
→ Create users using setup-users.html first

Problem: "Permission denied" in Firestore
→ Check Firestore security rules are set correctly

Problem: Can't create users
→ Make sure Firestore is enabled in Firebase Console

More help: See FIREBASE_SETUP.md
